Overview of Web Technologies

  • Browsers (front end / client) software programs people use to consume the web (Chrome, Safari, Firefox, etc…).
  • HTTP (Hypertext Transfer Protocol) messaging protocol that allows web browsers to communicate with web servers (where web sites are stored).
  • HTML, CSS , JavaScript three technologies needed to build a website that runs in the browser.
  • Server-side languages and frameworks (back end or server-side) languages that run on the server before the result is then sent to the browser to be displayed.

Study and Portfolio Tips

Study

  • Sign-up for programming course work GEO 244, GEO 242, CIS 120, CIS 121, CIS 275
  • Take advantage of online resources the GIS programming Roadmap is a good place to start
  • Go slowly when learning to code focus on one trajectory at a time until you reach comprehension
  • Start learning general web development first, then interactive mapping After your first bootcamp, start with Leaflet.js or ArcGIS JS API

Portfolio

  • Github Student Developer Pack sign up, take advantage of the free resources
  • Post working examples Publish your work on Github (or CodeSandbox, CodePen etc...)
